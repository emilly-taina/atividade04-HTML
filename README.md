# Atividade 04 - HTML

# Site sobre Law & Order: SVU

Nome completo: Emilly Tainá da Silva Alves

Descrição:
Este projeto foi desenvolvido como atividade prática de HTML. O site apresenta informações sobre a série Law & Order: Special Victims Unit, com foco especial na personagem Olivia Benson. O projeto contém três páginas HTML, menu de navegação, imagens, vídeo do YouTube, links externos e uma folha de estilo CSS para melhorar a aparência visual.
